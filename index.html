MULTIPLICATION||STUDENT MODE||BEGINNER LEVEL

INSTRUCTIONS:

  1. Arrange the polynomial expression in descending order depending on the exponent
  2. If one term is missing (i.e Ax3 + Cx + D => Bx2 is missing) put 0 as its coefficient
  3. Get the coefficients of each term of the polynomial expressions
  4. Enter each coefficient in the corresponding cells
  5. The product will show at the bottom part of the multiplication of polynomial table.
  6. To find the final answer, get the numbers in the “Product” tab then add the literal coefficients by writing “x” in each numeral coefficients except the last coefficient. Add a descending exponent in every literal coefficients x.
  1. Arrange the polynomial expression(Dividend and Divisor) in descending order depending on the exponent
  2. If one term is missing (i.e Ax3 + Cx + D => Bx2 is missing) put 0 as its coefficient
  3. Get the coefficients of each term of the polynomial expressions
  4. Enter each coefficient in the corresponding cells.
    1. enter the divisor on first column on multiplication of polynomial table.
    2. Enter the dividend on Dividend Tab.
  5. To get the quotient;
    1. get the factor of the first coefficients, one coefficient is given in the divisor(column 1, row 2) the other one will determine by the solver.
    2. You can use(optional) the “Get Factor’’ tab to help you find the factor of the coefficients.
    3. The step continues until you use all the coefficients of the dividend
  6. You can use the “Guide” tab to determine the missing number in corresponding diagonal color. The sum of this diagonal color corresponds to the coefficients of the dividend.
  7. If the Guide tab turns to all zero number it means that you got the correct answer.
  8. To find the quotient, get the numbers in the first row of the multiplication of polynomial table then add the literal coefficients by writing “x” in each numeral coefficients except the last coefficient. Add a descending exponent in every literal coefficients x.
  1. Arrange the polynomial expression in descending order depending on the exponent.
  2. If one term is missing (i.e Ax3 + Cx + D => Bx2 is missing) put 0 as its coefficient.
  3. Get the coefficients of each term of the polynomial expressions.
  4. Enter each coefficient in the "Dividend" column.
  5. Set the highest degree of exponent based on the problem given.
  6. To get the factor of the polynomial;
    1. get the factor of the last coefficients.
    2. You can use the “Get Factor" tab to help you find the factor of the coefficients.
    3. Use each number of the factors obtained in Get factor tab, kindly enter each number “x = “ tab, click “check”. Once you obtained a “0” value in the checker, it means that number is one of the factor of a given polynomial expression.
  7. After you determine one factor of the polynomial, you can use the “multiplication of polynomial table” to find the other factor. Kindly enter in the first column the corresponding coefficients(don't forget to equate it to 0) of one factor(i.e. “x - 3 = 0” , use 1 and -3 as the coefficients).
  8. Use the last coefficients as the starting point, look for the other factor in the “Get factor tab” that is paired in the corresponding number in the divisor(other factor).
  9. Continue until your fill all the boxes in the multiplication of polynomial table.
  10. If the Guide tab turns to all zero number it means that you got the correct answer.
  11. To find the factors, get the numbers in the first row and first column of the multiplication of polynomial table then add the literal coefficients by writing “x” in each numeral coefficients except the last coefficient. Add a descending exponent in every literal coefficients x.
